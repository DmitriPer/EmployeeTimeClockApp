Description
Implement a web-based Time Clock. The Time Clock should allow an employee to login using his/her ID
Number and log entry/break/exit throughout the day.
Once logged in, the employee is able to see a tabular view of previous entries/exits per day.
The employee may enter a note next to each entry and also export a PDF/CSV/XLS of the entries.
